# Landing Page Tele Severino

Tela antes do login, seguindo padrão visual clean/premium, inspirado em páginas modernas como Apple/GitHub.

## Conceito principal

Conhecimento na hora certa.

Conecte usuários a especialistas para resolver dúvidas por vídeo, voz ou mensagem, com atendimento simples, rápido e confiável.

## Público usuário comum

- Usa a plataforma gratuitamente
- Pesquisa especialistas
- Paga apenas pelo atendimento realizado
- Pode buscar por categoria, ramo, tags, avaliação e tempo médio de atendimento

## Público especialista

- Assina a plataforma como SaaS
- Cria perfil profissional
- Escolhe ramos de atuação
- Recebe por minuto
- Pode usar cupons iniciais para atrair clientes
- Pode contratar planos maiores para mais ramos, cupons e visibilidade

## Seções da landing page

- Topo com logo, links e botões Entrar / Criar conta
- Hero com título "Conhecimento na hora certa."
- Botões "Começar agora" e "Quero ser especialista"
- Cards rápidos:
  - Atendimento por minuto
  - Especialistas verificados
  - Categorias flexíveis
  - Assinatura para especialistas
- Mockup da plataforma
- Benefícios:
  - Para usuários
  - Para especialistas
  - Assinatura flexível
  - Atendimento inteligente
